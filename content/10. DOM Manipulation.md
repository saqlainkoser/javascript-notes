# DOM Manipulation вЂ” Complete Guide
## 1. DOM Introduction

The **DOM (Document Object Model)** is a tree-like representation of an HTML page that JavaScript can read and modify. Every HTML tag becomes a **node/object** in this tree, so JavaScript can access, change, add, or remove content dynamically.

```html
<body>
  <h1 id="title">Hello</h1>
</body>
```

```javascript
console.log(document);
```

The `document` object is the entry point to the entire DOM tree.


## 2. Selecting Elements

Before you can manipulate anything, you need to **select** the element first.

|Method|Returns|Live?|
|---|---|---|
|`getElementById`|single element|вЂ”|
|`getElementsByClassName`|HTMLCollection|live|
|`getElementsByTagName`|HTMLCollection|live|
|`querySelector`|single element (first match)|вЂ”|
|`querySelectorAll`|NodeList|static|

---

## 3. `querySelector`

Selects the **first** element that matches a CSS selector.

```html
<p class="msg">Hello</p>
<p class="msg">World</p>
```

```javascript
const el = document.querySelector(".msg");
console.log(el.textContent);
```

**Output**

```
Hello
```

Works with any CSS selector: `#id`, `.class`, `tag`, `[attribute]`, etc.

```javascript
document.querySelector("#title");
document.querySelector("div.card > p");
document.querySelector("input[type='text']");
```

---

## 4. `querySelectorAll`

Selects **all** matching elements and returns a static **NodeList**.

```html
<li class="item">A</li>
<li class="item">B</li>
<li class="item">C</li>
```

```javascript
const items = document.querySelectorAll(".item");
console.log(items.length);

items.forEach(item => console.log(item.textContent));
```

**Output**

```
3
A
B
C
```

> NodeList supports `forEach`, but it is **not** a real array (no `map`/`filter` directly вЂ” convert with `Array.from(items)` if needed).

---

## 5. `getElementById`

Selects a single element by its unique `id`. Fastest selection method.

```html
<h1 id="title">Welcome</h1>
```

```javascript
const heading = document.getElementById("title");
console.log(heading.textContent);
```

**Output**

```
Welcome
```

## 6. Modifying Text and HTML Content

|Property|Use|
|---|---|
|`textContent`|plain text, safe, ignores HTML tags|
|`innerText`|plain text, respects CSS visibility|
|`innerHTML`|reads/writes HTML markup|

```javascript
const box = document.querySelector("#box");

box.textContent = "Plain text here";
box.innerHTML = "<strong>Bold text</strong>";
```

```javascript
// Reading
console.log(box.innerHTML);
```

> вљ пёЏ Avoid inserting untrusted user input with `innerHTML` вЂ” it can lead to **XSS** attacks. Use `textContent` for plain text.

## 7. Manipulating Styles

### Inline style

```javascript
const box = document.querySelector(".box");

box.style.color = "white";
box.style.backgroundColor = "black";
box.style.padding = "10px";
```

### Using classList (preferred for reusable styles)

```javascript
box.classList.add("active");
box.classList.remove("hidden");
box.classList.toggle("dark-mode");
box.classList.contains("active"); // true / false
```

|Method|Purpose|
|---|---|
|`add()`|add a class|
|`remove()`|remove a class|
|`toggle()`|switch class on/off|
|`contains()`|check if class exists|

## 8. Creating and Removing Elements

### Create

```javascript
const li = document.createElement("li");
li.textContent = "New Item";

document.querySelector("ul").appendChild(li);
```

### Insert at specific position

```javascript
const ul = document.querySelector("ul");
const newItem = document.createElement("li");
newItem.textContent = "First Item";

ul.prepend(newItem);              // insert at start
ul.insertBefore(newItem, ul.children[1]); // insert before a specific child
```

### Remove

```javascript
const item = document.querySelector("li");
item.remove();               // modern way

// older way
item.parentElement.removeChild(item);
```

## 9. Traversing DOM Elements

Moving between related elements in the tree.

```javascript
const item = document.querySelector(".item");

console.log(item.parentElement);      // parent
console.log(item.children);           // child elements
console.log(item.nextElementSibling); // next sibling
console.log(item.previousElementSibling); // previous sibling
```

|Property|Gives|
|---|---|
|`parentElement`|direct parent|
|`children`|element children only|
|`childNodes`|all nodes (incl. text/comments)|
|`firstElementChild`|first child element|
|`lastElementChild`|last child element|
|`nextElementSibling`|next sibling element|
|`previousElementSibling`|previous sibling element|

## 10. Practical DOM Examples

### Example 1 вЂ” Change text on click

```javascript
const btn = document.querySelector("#btn");
const text = document.querySelector("#text");

btn.addEventListener("click", () => {
  text.textContent = "Text Changed!";
});
```

### Example 2 вЂ” Add list items dynamically

```javascript
const list = document.querySelector("#list");

function addItem(name) {
  const li = document.createElement("li");
  li.textContent = name;
  list.appendChild(li);
}

addItem("Apple");
addItem("Banana");
```

### Example 3 вЂ” Toggle a class

```javascript
const box = document.querySelector(".box");
document.querySelector("#toggleBtn").addEventListener("click", () => {
  box.classList.toggle("active");
});
```

## 11. Event Handling

An **event** is an action that happens in the browser (click, keypress, load, etc.). We "listen" for events using `addEventListener`.

```javascript
element.addEventListener("event-name", function (event) {
  // handler code
});
```

```javascript
const btn = document.querySelector("#btn");

btn.addEventListener("click", function () {
  console.log("Button clicked!");
});
```

You can attach **multiple** listeners to the same element, and remove them with `removeEventListener` (only works if the handler is a named function, not an anonymous one).

```javascript
function handleClick() {
  console.log("Clicked");
}

btn.addEventListener("click", handleClick);
btn.removeEventListener("click", handleClick);
```

## 12. Mouse Events

|Event|Fires when...|
|---|---|
|`click`|element is clicked|
|`dblclick`|element is double-clicked|
|`mouseover`|pointer enters the element|
|`mouseout`|pointer leaves the element|
|`mousedown`|mouse button pressed|
|`mouseup`|mouse button released|
|`mousemove`|pointer moves over element|

```javascript
const box = document.querySelector(".box");

box.addEventListener("mouseover", () => {
  box.style.backgroundColor = "yellow";
});

box.addEventListener("mouseout", () => {
  box.style.backgroundColor = "white";
});
```

---

## 13. Keyboard Events

|Event|Fires when...|
|---|---|
|`keydown`|a key is pressed down|
|`keyup`|a key is released|
|`keypress`|a key that produces a character is pressed (deprecated, avoid using)|

```javascript
document.addEventListener("keydown", (e) => {
  console.log("Key pressed:", e.key);
});
```

**Output (example, pressing "a")**

```
Key pressed: a
```

Useful properties: `e.key` (character), `e.code` (physical key), `e.shiftKey`, `e.ctrlKey`, `e.altKey`.

## 14. Form Events

|Event|Fires when...|
|---|---|
|`submit`|a form is submitted|
|`change`|an input's value changes and loses focus|
|`input`|an input's value changes instantly (every keystroke)|
|`focus`|element gains focus|
|`blur`|element loses focus|

```html
<form id="myForm">
  <input type="text" id="name" />
  <button type="submit">Submit</button>
</form>
```

```javascript
const form = document.querySelector("#myForm");
const input = document.querySelector("#name");

form.addEventListener("submit", (e) => {
  e.preventDefault();
  console.log("Submitted:", input.value);
});

input.addEventListener("input", (e) => {
  console.log("Typing:", e.target.value);
});
```

## 15. Event Object

Every event handler automatically receives an **event object** with details about the event.

```javascript
document.querySelector("#btn").addEventListener("click", function (event) {
  console.log(event.type);    // "click"
  console.log(event.target);  // the exact element that triggered it
});
```

|Property|Meaning|
|---|---|
|`event.type`|name of the event|
|`event.target`|the element that triggered the event|
|`event.currentTarget`|the element the listener is attached to|
|`event.key`|key pressed (keyboard events)|
|`event.clientX/Y`|mouse coordinates (mouse events)|

---

## 16. Prevent Default Behavior

Some elements have default browser behavior (form submission reloads the page, links navigate, etc). `preventDefault()` stops that.

```javascript
document.querySelector("a").addEventListener("click", (e) => {
  e.preventDefault();
  console.log("Link click blocked");
});
```

```javascript
form.addEventListener("submit", (e) => {
  e.preventDefault(); // stops page reload
  console.log("Handled with JS instead");
});
```
